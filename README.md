# AI 전문인재 양성과정 — 3D 시네마틱 브리핑

`index.html` 하나로 동작하는 **3D 스크롤 스크럽 + 자동재생 시네마틱** 페이지입니다.
브라우저에서 파일을 열면 한 편의 영상처럼 나레이션(TTS)과 자막이 흐르고,
스크롤하면 언제든 직접 장면을 앞뒤로 스크럽할 수 있습니다.

## 사용 방법

- `index.html`을 브라우저(크롬 권장)에서 엽니다. 서버·빌드·외부 라이브러리 불필요.
- **▶ 영상처럼 재생하기** — 나레이션·자막·카메라가 자동으로 진행됩니다.
- **스크롤** — 자동재생이 멈추고 스크럽 모드로 전환됩니다. `Space`로 현재 위치에서 이어보기.
- **🔊 나레이션** 버튼으로 음소거(자막만) 전환.

## 나레이션 보이스 (ElevenLabs ssjvoice)

나레이션은 아래 우선순위로 재생됩니다.

1. **`audio/s{장면}-l{문장}.mp3`** — 미리 구운 파일이 있으면 최우선 (열람자 키 불필요)
2. **ElevenLabs API 실시간 합성** — 페이지의 **🎙 보이스 연결** 버튼으로 API 키를 입력하면
   계정의 `ssjvoice` 보이스로 합성합니다. 키는 해당 브라우저의 localStorage에만 저장되고
   파일·리포에는 포함되지 않습니다. 합성 결과는 IndexedDB에 캐시되어 재생 시 재과금되지 않습니다.
3. **브라우저 내장 TTS** — 위 둘이 없을 때 폴백.

### mp3 미리 굽기 (배포용 권장)

```bash
ELEVENLABS_API_KEY=xi-... node scripts/bake-narration.mjs
```

`index.html`의 SCRIPT를 읽어 전 문장을 `audio/`에 mp3로 저장합니다.
이후에는 누구에게 공유해도 키 없이 ssjvoice 나레이션이 재생됩니다.
SCRIPT(대본)를 수정하면 다시 실행해 주세요.

## 내용(글) 교체하기

`index.html`의 `SCRIPT` 배열만 수정하면 됩니다. 장면 수는 자유입니다.

```js
{
  kicker : "장면 상단 영문 라벨",
  title  : "큰 제목 (\n 으로 줄바꿈)",
  sub    : "보조 문장 (생략 가능)",
  lines  : ["나레이션 문장 1", "문장 2 — <em>강조</em> 태그 사용 가능"],
  form   : "sphere | torus | helix | wave | tunnel | text",
  text   : "form이 'text'일 때 파티클로 그릴 짧은 단어",
  accent : "#e2e8f0"   // 장면 포인트 색 (모노톤 지침: 그레이스케일 사용)
}
```

## 3D 연출 구조

각 장면은 3D 우주 공간에 떠 있는 **정거장**(파티클 구조물: 구·토러스·나선·물결·터널·글자)이고,
카메라가 주인공 시점으로 **빛의 길**을 따라 정거장 사이를 비행합니다.

- **오빗 + 프레지 줌인** — 장면 동안 정거장을 궤도 선회하며 서서히 다가감
- **뱅킹 트랜짓** — 장면 전환 시 기울며 선회 비행, FOV 킥(워프감), 먼지 스피드라인
- **웨이포인트 비콘** — 다음 목적지에 맥동하는 링 (게임 내비게이션 느낌)
- **공간 부착 타이틀 카드** — 화면 고정이 아니라 정거장 위 공간에 붙어 있어 다가가면 커짐
- 스크롤 텍스트 리빌(단어 스태거), 레터박스·자막, `prefers-reduced-motion` 대응, 의존성 0
